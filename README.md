# Freelancer Business Activities Analysis Tool

## Overview
The Freelancer Business Activities Analysis Tool is designed to provide detailed insights into various freelance activities licensed in Abu Dhabi. Using this tool, users can analyze trends, identify market gaps, and make data-driven decisions to support the development of the freelance economy.

## Features
- **Interactive Visualizations**: Explore trends and patterns in freelancer business activities.
- **Bilingual Support**: Works seamlessly in both English and Arabic.
- **Filtering and Exporting**: Filter data by categories and export filtered data for further analysis.
- **Guided Usage**: Includes tutorials and sample queries to help users get started.

## Requirements
- Python 3.7+
- Pandas library
- Matplotlib library

## Installation
1. Clone the repository:
   bash
   git clone https://github.com/your-repo/Freelancer-Activities-Analysis.git
   
2. Navigate to the project directory:
   bash
   cd Freelancer-Activities-Analysis
   
3. Install required dependencies:
   bash
   pip install pandas matplotlib
   

## Usage
1. Place the dataset file (`DL06-Freelance-Activities-ADRA-OD-010-AFR.csv`) in the project directory.
2. Run the script:
   bash
   python analysis_tool.py
   
3. View the output visualizations and exported filtered data.

## Example Queries
- What are the most common freelance activities in Abu Dhabi?
- How many consultancy services are listed in the dataset?
- Which activities are related to sports and recreation?
